## Where to enter (Play Console)
**Grow → Store presence → Store listings (Main store listing)** → button **«Управление переводами» / Manage translations** → **Добавить переводы → Добавить свой перевод** → tick the 4 languages above → **Применить**.
Then a **language selector** appears at the top of the Main store listing page. Switch to each language and paste **App name / Short description / Full description** below. Leave graphics blank → they inherit the default (English) icon/feature-graphic/screenshots (fine for launch).

Limits: App name ≤30 · Short ≤80 · Full ≤4000.
